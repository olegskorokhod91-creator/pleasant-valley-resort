# BRANDGUIDELINES.MD — VISUAL & DESIGN DIRECTION

This website must feel like:
- a premium boutique resort
- a high-end Airbnb experience
- something that looks expensive and intentional

NOT:
- a template site
- not cluttered
- not overly flashy

---

# CORE FEEL

The user should feel:
- calm
- impressed
- slightly luxurious
- inspired to book a group stay or event

---

# VISUAL STYLE

## Colors
- soft whites (#FAF9F6)
- warm wood tones
- muted sage greens
- charcoal accents

Avoid:
- harsh contrasts
- bright cheap colors

---

## TYPOGRAPHY

- large, elegant headings
- lots of spacing
- minimal text blocks
- premium, modern fonts

---

## IMAGERY

- large, cinematic images
- warm lighting
- natural tones
- lifestyle-driven

Images should:
- feel intentional
- not random
- not repetitive

---

# LAYOUT PHILOSOPHY

- image-first design
- clean spacing
- no clutter
- sections should breathe

Use:
- full-width sections
- layered visuals
- subtle transitions

---

# INTERACTIONS

- smooth animations (0.2–0.4s)
- subtle hover effects
- cards slightly lift on hover
- images softly zoom

NO:
- aggressive animations
- flashy effects

---

# GALLERY EXPERIENCE

Homepage gallery must feel:
- editorial
- curated
- premium

NOT:
- basic slideshow

Preferred:
- cinematic carousel
- smooth transitions
- focus on one image at a time

---

# BACKGROUNDS

Images can be used as backgrounds BUT:

- ALWAYS protect readability
- use overlays (dark/gradient)
- never place text directly on busy images

---

# BUTTONS / CTA

- clear
- visible
- high contrast

Must feel:
- clickable
- important

Avoid:
- blending into background
- low contrast

---

# FEELING TO ACHIEVE

This site should feel like:
“Someone spent serious money designing this.”

Users should think:
- this place is premium
- this is perfect for a group trip
- this would be amazing for a wedding

---

# FINAL RULE

Everything should feel:
- intentional
- balanced
- high-end
- simple but powerful

No element should feel accidental.

---

END GOAL:
A visually stunning, premium hospitality website that makes users want to book immediately.
